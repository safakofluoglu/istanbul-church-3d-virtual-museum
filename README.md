# Istanbul Church 3D Virtual Museum

An immersive 3D virtual museum experience showcasing the architecture and history of a historic church in Istanbul.

## Tech Stack
- React
- TypeScript
- Three.js
- React Three Fiber
- Vite

## Features
- Immersive 3D panorama navigation
- Interactive hotspots
- Section-based exploration
- Clean museum-grade UX

## Local Setup
```bash
npm install
npm run dev
```
